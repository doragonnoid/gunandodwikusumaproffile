# Design Notes — Outreach-inspired Direction

Pola yang diterapkan dari referensi bersifat konseptual, bukan penyalinan:

- Scrollytelling berbasis chapter
- Presentasi objek utama sebagai pusat narasi
- Data labels dan telemetry sebagai elemen visual
- Tipografi serif skala besar dengan outline treatment
- Instruksi interaksi langsung di layar
- Audio toggle dan atmosfer visual
- Full-screen editorial project sequences
- Navigasi minimal dan progres kontekstual

Identitas akhir menggunakan nama, foto, model virtual, proyek, dan narasi asli Gunando Dwi Kusuma.

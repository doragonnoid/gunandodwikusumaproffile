# Fitur yang Diimplementasikan

## GDK Professional Portfolio

Beranda diubah menjadi pengalaman portofolio berbasis narasi dengan lima chapter:

1. **Origin** — hero, objek identitas interaktif, video, dan audio.
2. **Identity** — profil Gunando Dwi Kusuma dan identitas Doragonnoid.
3. **Capabilities** — kompetensi dalam format accordion dan telemetry.
4. **Projects** — empat proyek utama dalam panel full-screen bertumpuk.
5. **Transmission** — filosofi kerja dan call-to-action kontak.

## Interaksi utama

- Preloader persentase
- Canvas starfield
- Tekstur visual noise
- Objek dua sisi yang dapat diputar dengan mouse atau touch
- Chapter rail dengan active state
- HUD nomor chapter
- Audio ambient manual
- Full-screen menu
- Contact modal
- Scroll progress
- Reveal animation dan parallax
- Capability accordion
- Animated counters
- Pointer depth pada kartu proyek
- Keyboard navigation menggunakan Arrow Up/Down dan Page Up/Down
- Transisi halaman internal
- Custom cursor

## Responsivitas dan aksesibilitas

- Layout desktop, tablet, dan mobile
- Navigasi chapter disederhanakan pada layar kecil
- Touch interaction untuk objek hero
- Dukungan `prefers-reduced-motion`
- Fallback `<noscript>` agar konten tetap terlihat tanpa JavaScript
- Semantic sections, labels, dan ARIA pada elemen interaktif

## Halaman proyek

- Relaxplace Doragonnoid — Roblox game
- Reflection — Android application
- Simple Animation Blender — 3D animation
- Backroom End — VR dan IoT

Semua aset visual tetap menggunakan karya dan dokumentasi milik Gunando Dwi Kusuma.

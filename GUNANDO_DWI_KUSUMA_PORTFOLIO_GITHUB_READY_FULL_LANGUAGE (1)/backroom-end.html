BackroomEnd

Prototipe VR berbasis Unity yang menghubungkan pergerakan, interaksi objek, dan controller virtual dengan sistem IoT untuk mengendalikan aktuator serta menampilkan data sensor.

project overview

Menghubungkan ruang virtual dengan perangkat nyata.

Backroom End adalah proyek eksperimental Virtual Reality dan Internet of Things yang dibangun di Unity menggunakan sistem XR.

Pemain menjelajahi lingkungan laboratorium futuristik melalui teleportation dan snap turn, menggunakan aset tangan kiri dan kanan, serta berinteraksi dengan objek fisika seperti bola basket. Sistem VR juga dirancang untuk mengirim perintah ke perangkat IoT agar dapat menyalakan LED, buzzer, atau servo. Sebaliknya, data suhu dari sensor DHT11 diterima dan ditampilkan kembali pada monitor di dalam ruang virtual.

UnityVirtual RealityXR Interaction ToolkitIoTSensor DataAndroid XR
vr features

Pergerakan dan interaksi yang dirancang untuk pengalaman XR.

Fitur VR berfokus pada locomotion yang nyaman, representasi tangan, interaksi fisika, dan komunikasi dua arah dengan sistem IoT.

01

Snap Turn

Pemain dapat memutar arah pandang dalam sudut tertentu melalui controller, sehingga navigasi VR terasa cepat dan mengurangi kebutuhan rotasi fisik.

02

Teleportation

Sistem teleportation memungkinkan pemain berpindah ke titik yang valid pada area permainan tanpa harus berjalan terus-menerus di ruang nyata.

03

Left & Right Hand Assets

Aset tangan kiri dan kanan merepresentasikan controller pengguna di dunia virtual dan menjadi dasar untuk memilih serta memegang objek.

04

Hand Model Animation

Model tangan merespons input trigger, grip, dan thumbstick sehingga gerakan controller terlihat lebih hidup dan komunikatif.

05

Grab & Throw

Objek yang memiliki collider, rigidbody, dan XR Grab Interactable dapat diambil, dipindahkan, lalu dilempar menggunakan gerakan tangan VR.

06

VR–IoT Messaging

Sistem menghubungkan VR dengan perangkat IoT untuk mengirim perintah, menerima respons, dan menampilkan status perangkat pada monitor virtual.

iot features

Perintah virtual menghasilkan aksi pada perangkat fisik.

Tiga aktuator dapat dikendalikan dari VR, sementara satu sensor mengirimkan data kembali untuk divisualisasikan di dalam aplikasi.

07

LED Control

Tombol di ruang virtual mengirim data ke perangkat IoT untuk menyalakan atau mematikan LED sebagai indikator visual.

08

Buzzer Control

Perintah dari VR dapat mengaktifkan buzzer untuk menghasilkan respons suara pada perangkat IoT.

09

Servo Control

Data yang dikirim dari VR digunakan untuk mengubah posisi atau sudut servo sesuai aksi yang dipilih pengguna.

10

DHT11 Sensor Display

Data suhu dari sensor DHT11 dikirim oleh sistem IoT dan ditampilkan pada panel atau monitor virtual agar dapat dibaca langsung dari VR.

system workflow

Komunikasi dua arah dari VR ke IoT dan kembali lagi.

Input controller diproses Unity, dikirim sebagai pesan ke perangkat IoT, menjalankan aktuator, kemudian data sensor dikirim kembali untuk ditampilkan di dunia virtual.

Controller input→Unity XR interaction→Send IoT message→LED · buzzer · servo→DHT11 sensor→Display in VR
